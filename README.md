# Installation:
### Ensure for all launchers that the modpack is given >6 GB (6144 MB) of RAM. 8GB is preferred.


### Releases page on this repository contains all files required for installation. 


## Prism/General Launcher (Recommended)
(Prism Launcher is available at https://prismlauncher.org/)

1. Download "Apple Tree Pack vX.X.Xx GENERAL" .zip file from releases

2. Add an instance
<img width="512" height="344" alt="prism1" src="https://github.com/user-attachments/assets/66aef053-5de6-4508-8596-cfc892339fe7" />

3. "Import", then "Browse" to select the downloaded modpack .zip file
<img width="662" height="476" alt="prism2" src="https://github.com/user-attachments/assets/f17ab672-1510-4a80-8de5-1f62ce009cd9" />

4. Finish import!


## Modrinth Launcher/App

1. Download "Apple Tree Pack vX.X.Xx MODRINTH" .mrpack file from releases

2. Add an instance using the "+" icon
<img width="549" height="617" alt="modrinth1" src="https://github.com/user-attachments/assets/b2912a61-e682-4998-bc4d-ef49a517dc02" />

3. Import from file, and select the downloaded modpack .mrpack file
<img width="601" height="346" alt="modrinth2" src="https://github.com/user-attachments/assets/fbcebec1-28e6-4416-94bd-95363fe5b941" />

4. Wait for download and finish import!


## Curseforge App (Not Recommended)
Curseforge is not recommended due to licensing/compatibility issues and also due to the Curseforge app just being terrible.
Issues will likely arise with Curseforge compatability in the future, and this pack may not support Curseforge forever.

1. Download "Apple Tree Pack vX.X.Xx CURSEFORGE" .zip file from releases

2. Ensure Minecraft Java Edition is selected, then import an instance
<img width="639" height="441" alt="curseforge" src="https://github.com/user-attachments/assets/3d90c884-706d-4b1b-9313-8e9e19754335" />

3. Import from .zip file and select the downloaded modpack .zip file
<img width="679" height="512" alt="curseforge2" src="https://github.com/user-attachments/assets/b681daa3-6384-48b1-bcbc-7b4b35b29762" />
